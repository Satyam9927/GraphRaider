Architecture

The Three-Agent Framework

Every test runs through a pipeline of three specialized agents - ensuring reliable verdicts even on flaky or slow endpoints.

Verdicts

Three evaluation modes

From fully-offline deterministic rules to Claude-assisted analysis - choose the mode that fits your environment.

Rule-Based
No API key
Hybrid
Rules + Claude
Full Claude
AI-powered

Graceful fallback - Hybrid and Full Claude modes fall back to rule-based if the LLM call fails.  Model: claude-sonnet-4-6

Authentication

Flexible dual-session auth

Two independent sessions let you run cross-session and BOLA authorization tests - each with its own auth method.

Quick Start

Up in 60 seconds

One script installs everything, launches the backend and frontend, and opens the browser.

# 1. Clone the repo
git clone https://github.com/Satyam9927/GraphRaider.git
cd GraphRaider
# 2. Make scripts executable and run
chmod +x start.sh kill.sh
./start.sh
