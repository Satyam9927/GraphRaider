When enabled, every request from the Runner and Repeater is sent through the proxy below, and TLS verification is disabled so Burp Suite / OWASP ZAP can intercept.
Burp Suite / ZAP setup
1. Start the proxy listener on 127.0.0.1:8080.
2. Enable the toggle above and Save.
3. GraphRaider disables certificate verification automatically while proxying — no CA import needed.
4. Every test-case and Repeater request appears in the proxy's HTTP history for manual review.
5. Use Send to Repeater from History to replay any captured request by hand.
